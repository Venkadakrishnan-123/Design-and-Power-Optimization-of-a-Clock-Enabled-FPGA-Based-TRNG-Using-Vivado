# Design-and-Power-Optimization-of-a-Clock-Enabled-FPGA-Based-TRNG-Using-Vivado
Designed and implemented a clock-enabled FPGA-based True Random Number Generator (TRNG) using Verilog HDL in Vivado to generate secure random numbers with improved energy efficiency. By reducing unnecessary switching activity through clock-enable control, the design achieved low power consumption (0.252 W) while maintaining high-quality randomness, making it suitable for cryptographic, IoT, and embedded security applications.
